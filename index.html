i 使用指南

使用指南

快速了解 Gain Staging、網站功能與 Simulator 操作方式。

1. 什麼是 Gain Staging？

Gain Staging（增益級距）是調整音訊系統中各個階段訊號電平的過程。本工具提供常見樂器與聲源的建議參考值，協助建立健康的輸入電平，降低失真風險，並保留足夠 Headroom。

  • 建議 RMS 範圍
  • 建議 Peak 範圍
  • 建議麥克風類型與推薦型號
  • Headroom 與使用注意事項

也可以透過 Gain Staging Simulator，理解 Gain 與 Fader 的不同作用。

2. 如何使用本工具

  1. Step 1選擇樂器分類。
  2. Step 2點選欲查詢的樂器或聲源。
  3. Step 3查看 RMS、Peak、Headroom、推薦麥克風與注意事項。
  4. Step 4點擊「前往模擬器」，載入目前樂器設定。

3. Gain Staging Simulator

Gain / Preamp
控制訊號進入系統的輸入電平。請先調整 Gain，使 Peak 落在建議範圍內。
Output Fader
控制輸出音量。Fader 只會改變輸出大小，不會修復已經發生的 Input Clip。
Input Meter
代表 Preamp 後的實際輸入電平，請依照建議 RMS 與 Peak 範圍調整 Gain。
Output Meter
代表經過 Fader 後送往主輸出的電平。

4. 快速操作

  • Gain Knob：點擊 🎯 按鈕，可快速回到目前樂器 / 聲源的建議 Gain。桌機版也可雙擊 Gain Knob 快速回到建議 Gain。
  • Output Fader：點擊 ↺ 按鈕，可快速回到 0 dB。桌機版也可雙擊 Output Fader 快速回到 0 dB。

5. 常見迷思

錯誤觀念：Gain 太大，把 Fader 拉低就能修復失真。

如果 Input 已經 Clip，失真已經在前級產生。即使降低 Fader，也無法修復已經失真的訊號。

  1. 先調整 Gain
  2. 確認 Input Peak 落在建議範圍
  3. 再使用 Fader 做輸出音量平衡

6. 注意事項

本工具提供的是現場音控的建議起點，而非固定標準。實際 Gain 會受到麥克風靈敏度、樂器音量、演奏方式、歌手音量、控台前級設計、PA 系統與演出環境影響。

請以耳朵與控台 Meter 為主要依據，本工具提供的是快速建立健康 Gain Structure 的參考值。
